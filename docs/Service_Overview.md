# サービスの概要

## サービス名（案）

**GMAP試験対策問題集アプリ**

## サービスコンセプト

GMAP試験の受験者が、スマートフォンやウェブブラウザを通じて、いつでもどこでも効率的に学習できる問題集アプリを提供します。

## ターゲットユーザー

| セグメント | 説明 |
|------------|------|
| **GMAP受験予定者** | 試験合格を目指し、問題演習を希望する学習者 |
| **再受験者** | 過去に不合格となり、再チャレンジする方 |
| **スキルアップ目的** | 資格取得以外に、知識習得を目的とする方 |

## 提供価値

1. **利便性**：スマホ・ウェブでスキマ時間に学習可能
2. **即時性**：解答後すぐに採点・解説を確認可能
3. **集中学習**：GMAP試験に特化した問題で効率的に学習

## 主要機能（想定）

### フェーズ1（モック段階）

- 問題一覧の表示
- 個別問題の解答
- 採点結果の表示
- 解説の表示（簡易）

### フェーズ2以降（本番開発時）

- ユーザー登録・ログイン
- 学習進捗の記録
- 苦手分野の分析
- 模擬試験モード
- オフライン対応（要検討）

## 利用シーン

- 通勤・通学中の電車内
- 昼休みのオフィス
- 自宅のPC前
- カフェなど外出先

## 競合・差別化

- GMAP専用問題集としての特化
- デジタルネイティブなUI/UX
- 段階的な機能拡張による継続的な価値提供
