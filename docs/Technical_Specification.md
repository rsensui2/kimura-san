# 技術仕様・実装計画

## フェーズ1：HTMLモック

### ゴール

- スマホ・ウェブで動作するHTMLモックの完成
- UI/UXの検証と関係者への説明資料として活用

### 技術スタック（モック）

| 項目 | 内容 |
|------|------|
| **マークアップ** | HTML5 |
| **スタイル** | CSS3（レスポンシブ対応） |
| **インタラクション** | JavaScript（Vanilla or 軽量ライブラリ） |
| **動作環境** | モダンブラウザ（Chrome, Safari, Edge等） |

### モックで実装する画面

1. **トップ/問題一覧画面**
2. **問題解答画面**
3. **採点結果画面**
4. **解説画面**

### 成果物

- 静的HTMLファイル一式
- サンプル問題データ（JSON or 埋め込み）
- 簡易README（ローカルでの起動方法）
