# GMAP試験対策問題集アプリ - 提案資料

木村社長向けの提案資料・モック・LPをまとめたリポジトリです。

## ポータルページ

**[https://rsensui2.github.io/kimura-san/](https://rsensui2.github.io/kimura-san/)** からすべての資料にアクセスできます。

GitHub Pages で公開済みです。

## 構成

| フォルダ | 内容 |
|----------|------|
| `docs/` | 提案ドキュメント（Markdown） |
| `lp/` | ランディングページ（HTML） |
| `mock/` | スマホアプリモック（HTML） |
| `notebooklm/` | NotebookLM で作成した動画・PDF・画像 |

## ドキュメントの推奨読了順序

1. [エグゼクティブサマリー](./docs/Executive_Summary.md)

3. 必要に応じて [ビジネスモデル](./docs/Business_Model.md)、[技術仕様](./docs/Technical_Specification.md)、[プロジェクトスケジュール](./docs/Project_Schedule.md)

## GitHub Pages での公開

1. リポジトリの **Settings** → **Pages**
2. **Source** で `Deploy from a branch` を選択
3. **Branch** で `main`（または `master`）を選択
4. **Folder** で `/ (root)` を選択
5. **Save** をクリック

数分後、`https://<username>.github.io/<repo-name>/` でポータルが公開されます。
